# Stratify AI - Project Demonstration Guide

Follow this guide to give a structured, professional live demo to your mentor.

---

### 1. The Hook (Landing Page)
- **Start at**: `http://localhost:5174/`
- **Show**: The scroll animations (AOS) and the hero section.
- **Talking Point**: "We built an 'Orchestrator' that doesn't just give one answer, but runs an entire agency of AI agents to solve content creation."

### 2. The Login & Onboarding
- **Action**: Log in using a test account (or Google Login).
- **Show**: The smooth transition to the dashboard.
- **Talking Point**: "Notice the focus on speed. We use JWT tokens to keep the user session secure while navigating between services."

### 3. The Core Value (Strategic Planner)
- **Action**: Create a new strategy.
- **Scenario**: Use a niche like "Solar Energy for Indian Homeowners".
- **Observe**: The loading state while the agents (Researcher/Copywriter) are working.
- **Talking Point**: "In the background, our Researcher agent is scraping current search trends, while the Copywriter drafting hooks specifically for this niche."

### 4. High-Value Output (Results View)
- **Action**: Switch between the **SEO Pack**, **Content Calendar**, and **Sample Posts**.
- **Show**: The **Expected ROI** card.
- **Talking Point**: "We've localized the results. Notice the ROI is projected in **₹ (Rupees)**, making it instantly relevant for an Indian consultant or creator."

### 5. Backend Intelligence (Admin Dashboard)
- **Action**: Switch over to the **Admin Dashboard**.
- **Show**: The "AI Intelligence" graph and token usage.
- **Talking Point**: "This is where we monitor the health of the project. We track the cost of every strategy generated vs. the token consumption of our Llama 3.3 models."

### 6. The Upgrade Flow (Premium Experience)
- **Action**: Click "Upgrade to Pro".
- **Show**: The pricing table and the **₹ symbol**.
- **Talking Point**: "We've built a tiered system. Free users get 3 strategies, while Pro users get unlimited access and real-time SEO data via SerpAPI."
