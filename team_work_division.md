# Stratify AI - Team Work Division (2-Person Team)

To ensure a smooth presentation to your mentor, here is how the work is divided between you and your teammate.

---

### 👤 Person A: AI Logic & Backend Engineering
**Role**: The "Brain" Architect
**Responsibilities**:
- **CrewAI Orchestration**: Built the multi-agent system (Researcher, Copywriter, SEO Analyst) that works together to generate strategies.
- **FastAPI Development**: Implemented the core REST API endpoints and state management.
- **Data Modeling**: Managed MongoDB and Redis integrations for fast strategy retrieval and session caching.
- **Security**: Configured JWT authentication and protected routes for Admin and Pro tiers.
- **AI Token Management**: Tracking token usage (Llama 3.3) for ROI calculation.

### 👤 Person B: UI/UX & Frontend Architecture
**Role**: The "Experience" Designer
**Responsibilities**:
- **Modern Dashboard Design**: Created the premium, glassmorphism-style UI using Tailwind CSS and React.
- **Interactive Visuals**: Integrated Recharts and CountUp for real-time ROI and performance visualization.
- **State Management**: Built the React context for Auth and global state to keep the UI in sync with the backend.
- **Localization**: Localized the entire application experience (including Rupee ₹ symbol integration) for the target market.
- **User Flow Optimization**: Designed the "New Strategy" wizard and the multi-tab Strategy Results view.

---

### 🎤 Who should explain what during the presentation?

| Section | Speaker | Focus |
| :--- | :--- | :--- |
| **Project Overview** | Person B | The "Big Idea" and who it's for. |
| **AI Strategy Generation** | Person A | CrewAI, Agents, and Llama 33 models. |
| **The Result (Posts/Calendar)** | Person B | How the creator uses the output to grow. |
| **Admin & Business Logic** | Person A | Cost tracking, token usage, and MongoDB. |
| **ROI & Future Scope** | Both | Localization (₹), the impact on business, and next steps. |
