# CHAPTER 7: SCREEN DESCRIPTIONS

Below are the descriptions for the 10 screenshots included in Chapter 7. You can copy and paste these directly under each corresponding image in your document.

### 1. Login Screen
This screenshot displays the secure entry point to the PlanvIx platform. The interface handles user authentication, ensuring that only authorized individuals can access the strategy generation tools and analytics. It features a clean, user-friendly design with options for secure credential entry, leveraging JWT tokens in the backend to maintain a safe and persistent user session.

### 2. Dashboard
The Dashboard serves as the central hub where users are greeted immediately after logging in. It provides a high-level overview of the user's account, displaying recent activity, current subscription status, and quick-access buttons. From here, users can seamlessly navigate to initiate a new content strategy or review their historical campaigns.

### 3. Strategy Planner Screen (First Half)
This image captures the top section of the Strategy Planner, which acts as the primary input interface. Here, users define their core marketing parameters, such as the specific niche, target audience, and overarching campaign goals. It features intuitive form fields where users can specify the tone and context before dispatching tasks to the multi-agent AI system.

### 4. Strategy Planner Screen (Second Half)
The lower section of the Strategy Planner displays the real-time output of the AI orchestration pipeline. This screenshot highlights the detailed results generated by the specialized agents (such as the SEO and Copywriter agents), including comprehensive content calendars, recommended hooks, and expected ROI projections. It allows users to review the complete, synthesized strategy in one unified view.

### 5. Subscription Page (Razorpay Popup)
This screen illustrates the platform's secure monetization integration. When a user opts to upgrade their plan, a Razorpay payment gateway popup appears seamlessly over the interface. It presents the subscription tiers and securely collects payment information without redirecting the user away from the application, ensuring a smooth checkout experience.

### 6. Payment Successfully Page (Razorpay Popup)
Following a transaction, this confirmation screen validates the successful payment within the Razorpay interface. It provides the user with visual confirmation that their payment was processed securely. In the background, this triggers the system to instantly update their account status, unlocking premium features and expanding their token quota.

### 7. Analytics Pro Page (First Half)
This screenshot shows the upper section of the advanced analytics dashboard, exclusive to Pro users. It focuses on overarching performance metrics, displaying high-level data visualizations and graphs. Users can monitor token consumption, track the volume of strategies generated, and assess overall system usage at a glance.

### 8. Analytics Pro Page (Second Half)
The lower section of the Analytics Pro dashboard dives into more granular, actionable data. This interface provides detailed breakdowns of specific campaigns and tracks the projected ROI of individual strategies. It serves as a critical operational tool, allowing marketers to analyze localized insights and optimize their future content generation.

### 9. Strategy Planner Pro Page
This image showcases the enhanced version of the standard planner available to premium subscribers. The Pro interface includes advanced configurations, such as real-time SEO data integration (e.g., via SerpAPI) and unlimited usage options. It provides users with more nuanced controls over the AI agents, enabling the creation of highly specialized, data-backed marketing campaigns.

### 10. Admin Dashboard
The Admin Dashboard is a specialized control panel accessible only to platform administrators. This interface provides comprehensive backend intelligence, allowing admins to track the global health of the application. It features monitors for overall LLM token consumption, aggregate user activity, and system costs, ensuring smooth, scalable, and financially viable operations.
